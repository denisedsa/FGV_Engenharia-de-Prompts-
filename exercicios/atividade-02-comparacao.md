# Atividade 02 - Comparação de Prompts

## Objetivo

Comparar diferentes formas de estruturar uma solicitação para Inteligência Artificial, observando como pequenas mudanças no prompt podem influenciar a qualidade das respostas geradas.

---

## Cenário

Criar um resumo profissional para apresentação em processos seletivos.

---

## Prompt 01 - Solicitação simples

Solicitação realizada:

> Crie um resumo profissional.

### Observação

O resultado apresentou uma resposta mais genérica, pois não foram informados detalhes sobre o perfil profissional, objetivo do texto ou características esperadas.

---

## Prompt 02 - Solicitação estruturada

Após analisar a primeira resposta, foram adicionadas informações para direcionar melhor a IA.

### Prompt estruturado

> Atue como um profissional de recrutamento e seleção.
>
> Contexto:
> Preciso criar um resumo profissional para utilização em processos seletivos.
>
> Objetivo:
> Elaborar um texto que apresente experiência profissional, competências e principais características do perfil.
>
> Formato esperado:
> Criar um resumo objetivo, profissional e adequado para um currículo.

---

## Comparação dos resultados

| Critério | Prompt simples | Prompt estruturado |
|---|---|---|
| Contexto | Não informado | Contexto apresentado |
| Objetivo | Genérico | Definido |
| Direcionamento | Limitado | Mais específico |
| Resultado | Resposta ampla | Resposta mais alinhada |

---

## Resultado observado

A comparação demonstrou que a segunda versão do prompt apresentou uma resposta mais próxima do objetivo esperado, pois a IA recebeu informações adicionais para interpretar a necessidade.

---

## Aprendizado

A atividade reforçou que a qualidade da resposta depende não apenas da ferramenta utilizada, mas também da forma como a solicitação é estruturada.

A comparação entre diferentes abordagens ajuda a identificar quais elementos tornam um prompt mais eficiente.
