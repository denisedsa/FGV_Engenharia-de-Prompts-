# Aprendizados durante os estudos

## Principais percepções

Durante minha evolução no estudo de Engenharia de Prompts, identifiquei que criar bons prompts envolve mais do que fazer perguntas para uma IA.

É necessário compreender:

- Qual resultado desejo obter;
- Quais informações a IA precisa receber;
- Como organizar a solicitação;
- Como avaliar e melhorar a resposta.

---

## Processo de evolução

Minha prática tem seguido algumas etapas:

1. Criar uma primeira versão do prompt;
2. Avaliar a resposta gerada;
3. Identificar pontos de melhoria;
4. Ajustar a instrução;
5. Comparar os resultados.
